# windows system manager

- a windows system manager made in python
- has a gui
- depends on psutil

# how to use

1. install python 3 from https://python.org/downloads if you haven't yet
2. install psutil with the command "pip install psutil" (MAKE SURE PIP IS IN YOUR PATH!!!)
3. run the exe
